# 🚀 PromptCraft AI

PromptCraft AI is a full-stack AI-powered prompt enhancement application that transforms simple user prompts into detailed, structured, and effective prompts for Large Language Models (LLMs) like Gemini and ChatGPT.

---

## 🌐 Live Demo

**Link:** https://promptcraft-ai-skz2.onrender.com/

---

## 📌 Features

- ✨ AI-powered prompt enhancement using Google's Gemini API
- 🎯 Multiple enhancement modes
  - Professional
  - Coding
  - Creative
  - Marketing
  - Learning
- 📋 Copy enhanced prompt with one click
- 🕒 Prompt history stored using Local Storage
- 🗑️ Clear prompt and history functionality
- ⚡ Loading state while AI generates responses
- ❌ Error handling for failed API requests
- 📱 Responsive and modern UI
- 🎨 Smooth animations with Framer Motion

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/TanishaSharma10/PromptCraftAI.git
```

Go into the project

```bash
cd PromptCraftAI
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
